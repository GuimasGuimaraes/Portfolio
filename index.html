Portfólio de Testes Automatizados

Guilherme Guimarães

Sou um profissional especializado em testes automatizados com experiência em diversas ferramentas e frameworks. Meu objetivo é garantir a qualidade e a eficiência dos sistemas através de testes rigorosos e bem estruturados.

Projetos

Projeto 1: Testes Automatizados com Selenium

Desenvolvi um conjunto de testes automatizados para um site de e-commerce utilizando Selenium WebDriver. Os testes cobrem funcionalidades críticas como login, busca de produtos e finalização de compra.

Projeto 2: Integração Contínua com Jenkins

Implementei um pipeline de integração contínua utilizando Jenkins para automatizar a execução de testes em um projeto de desenvolvimento de software. Isso garantiu que os testes fossem executados automaticamente a cada commit.
